# Waveshare 1.3" ST7789 Wiring

The Waveshare 1.3 inch LCD HAT uses the ST7789 controller and communicates over SPI.

## Pin connections (Pi Zero 2 W)

| Display pin | Pi pin (BCM) | Notes                       |
|-------------|--------------|-----------------------------|
| VCC         | 3.3 V (pin 1 or 17) | Power for LCD             |
| GND         | GND (pin 6)         | Ground                    |
| SCL (CLK)   | GPIO 11 (pin 23)    | SPI SCLK                  |
| SDA (DIN)   | GPIO 10 (pin 19)    | SPI MOSI                  |
| DC          | GPIO 25 (pin 22)    | Data/Command select       |
| RST         | GPIO 27 (pin 13)    | Reset                     |
| CS          | GPIO 8  (pin 24)    | SPI chip select           |
| BL          | GPIO 18 (pin 12)    | Backlight control (optional) |

Ensure the SPI interface is enabled via `sudo raspi-config` → Interface Options → SPI.

The screen is rotated 270 degrees by default (landscape). Adjust the rotation parameter in the code if necessary.
